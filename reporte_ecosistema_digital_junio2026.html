Problemas Detectados & Soluciones

Frecuencia 4.94 — Fatiga creativa inminente en RMK Ventas Web

La misma persona ve el anuncio casi 5 veces. Benchmark recomendado para remarketing: máximo 3.0. Por encima de eso el CPM sube, la audiencia oculta el anuncio y el CPA empeora.

💡 Solución: Crear nuevo reel o imagen de CAVA esta semana. Rotar mínimo 3 creativos diferentes. Considerar ampliar la audiencia de remarketing a 60–90 días para bajar frecuencia.
📉

Pixel subreporta ventas — 57 Meta vs 75 reales (18 ventas sin atribuir)

Los registros de admin confirman 75 inscritos. El pixel solo registró 57 compras. Hay 18 ventas (24% del total) que no fueron atribuidas a las campañas. Esto significa el ROAS real es 43.4x, no el 69x que reporta Meta. Las causas más comunes: conversiones via WhatsApp, compras directas, o ventana de atribución vencida.

💡 Solución: Implementar event_id en GTM para deduplicación y mejorar cobertura del pixel. Activar la API de conversiones (CAPI) del servidor para capturar ventas que el pixel del navegador pierde. Esto subirá el ROAS reportado en Meta y mejorará la optimización de las campañas.
💬

WhatsApp: 59 conversaciones — solo 3 checkouts (5% conversión)

La campaña genera conversaciones pero sin ManyChat activo no hay seguimiento automatizado. Las personas llegan y se pierden sin secuencia de nutrición.
