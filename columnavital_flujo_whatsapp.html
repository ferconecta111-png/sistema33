Instrucciones para enviar la difusión en FunnelChat

Paso a paso

  1. Entra a FunnelChat → Transmisiones o Difusiones
  2. Haz clic en Nueva transmisión
  3. En filtro de contactos, selecciona: Sin etiqueta "agendado" + Sin etiqueta "consultado"
  4. Agrega también filtro por fecha: contactos que escribieron en los últimos 180 días
  5. Escoge el mensaje de difusión (Versión A, B o C — prueba las 3 en grupos distintos)
  6. Programa el envío para un martes, miércoles o jueves entre 9am–11am (mejor tasa de apertura)
  7. Activa el flujo de respuesta para que cuando alguien responda "SÍ" entre al Flujo 1 automáticamente

Configuración en FunnelChat — paso a paso

1. Crear etiquetas primero

Ve a Configuración → Etiquetas y crea estas:

  • nuevo-lead — se aplica cuando entra el primer mensaje
  • sincelejo / monteria / barranquilla / cartagena / valledupar / virtual
  • quiere-agendar — respondió que sí quiere cita
  • agendado — cita confirmada
  • consultado — ya tuvo su consulta
  • lead-frio — no respondió después de 2 seguimientos

2. Crear Flujo 1 — Bienvenida
